# IHES Anchor Mathematicians — Major Models & Formulas

Reference data for award-winning mathematicians affiliated with [Institut des Hautes Études Scientifiques (IHES)](https://www.ihes.fr/), Bures-sur-Yvette, France.

**Source:** [TANAAKK — Institut des Hautes Études Scientifiques｜IHES](https://www.tanaakk.com/2025/03/29/institut-des-hautes-etudes-scientifiques-ihes/)

This folder documents major formulas and models by Fields Medal and Abel Prize winners, ordered chronologically by publication year. Used for brushing up the L0–L4 layer theory.

- `CHRONOLOGICAL_INDEX.md` — All entries in chronological order
- Individual files per mathematician (e.g., `rene_thom.md`)
