# IAS Anchor Mathematicians — Major Models & Formulas

Reference data for mathematicians affiliated with the [Institute for Advanced Study (IAS)](https://www.ias.edu/) School of Mathematics, Princeton, USA.

**Source:** [TANAAKK — プリンストン高等研究所（IAS）数学部門](https://www.tanaakk.com/2025/04/20/ias-institute-for-advanced-study/)

This folder documents major formulas and models by IAS Mathematics faculty and emeriti, ordered chronologically by publication year. Complements the IHES anchor mathematicians reference.

## Contents

- `CHRONOLOGICAL_INDEX.md` — All entries in chronological order
- Individual files per mathematician (e.g., `jacob_lurie.md`)
